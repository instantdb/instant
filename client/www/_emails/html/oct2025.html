Guest Authentication

Instant now supports guest authentication! This allows your users to try your app before signing up and ensures they can keep all their data when they decide to create a full account with their email.

Use db.auth.signInAsGuest() to create a new guest user. This will create a new guest user with an id, but no email address.

When a guest user is ready to create a permanent account, you can use any of Instantâ€™s sign-in methods. The guest user will be automatically upgraded to a full user.
