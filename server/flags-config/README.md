Stores the perms and schema info for the flags in the instant-config app.

Install deps:
```
pnpm i
```

login:
```
pnpm run login
```

or in dev:
```
pnpm run dev:login
```

pull schema and perms:
```
pnpm run pull
```

or in dev:
```
pnpm run dev:pull
```

push schema and perms:
```
pnpm run push
```

or in dev:
```
pnpm run dev:push
```
